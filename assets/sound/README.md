# assets/sound（音声フォルダ）

効果音・BGMを入れる場所です。

## 形式
- **MP3 推奨**（Android・iPhone含め全対応・軽い）。
- 効果音は1秒以内目安、BGMは数MB以内目安。

## 例（ファイル名は分かりやすく）
- `chop.mp3` … 木を切る音
- `coin.mp3` … お金が入る音
- `sell.mp3` … 販売音
- `upgrade.mp3` … 強化・購入音
- `bgm.mp3` … 背景音楽（ループ）

## メモ
- スマホは最初のタップ後にしか音を鳴らせない仕様のため、BGMは初回タップで再生開始する形にします（実装側で対応）。
- オフライン再生用のキャッシュ登録も実装時に対応します。
- 音声は聴けないので、どれが何の音かを一言添えてください。
