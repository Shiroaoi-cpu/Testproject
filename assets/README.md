# assets（素材フォルダ）

ゲームで使う画像・音声を入れる場所です。

## 画像（PNG）
- **背景透過のPNG**をここ（`assets/`）か `assets/img/` に入れてください。
- キャラは、動かしたいなら複数コマ（立ち／振り上げ／振り下ろし／歩き など）を、各同じサイズ・位置揃えで。
- 例：`hero_idle.png`, `hero_chop1.png`, `hero_chop2.png`, `worker_walk1.png` …
- 高解像度（縦512pxくらい）推奨。表示時に縮小して使います。

## 音声（MP3）
- `assets/sound/` に入れてください（詳細は `sound/README.md`）。

## 渡すとき
- 入れたら「入れたよ」と一声かけてください。
- 画像は中身を確認できますが、音声は聴けないので、ファイル名か説明で「何の音／何のポーズか」を添えてください。
