# 木こりの森 — ゲーム設計書 v1

確定方針：**共通XP方式 / 4系統は相互依存 / 転生は後回し**

---

## 1. 共通XPとレベル

すべての行動で少量のXPが入り、単一のプレイヤーLvが上がる。Lvがアンロックの鍵。

### 行動ごとのXP（仮）
| 行動 | XP |
|---|---|
| 伐採（手動1回） | +1 |
| 加工・炭焼き | +2 |
| 販売 | +3 |
| 釣り上げ | +4 |
| 採掘 | +6 |
| 製錬 | +8 |
| 調理 | +12 |

- 自動化（木こり/削岩機等）で発生した行動も同じXPが入る（放置でも育つ）。
- 手動操作にはXP +50% ボーナス（能動プレイに旨味）。

### レベルカーブ（仮）
次のレベルに必要なXP：`xpToNext(Lv) = round(20 * Lv^1.7)`

| Lv | 次までのXP | 解放 |
|---|---|---|
| 1 | 20 | 材木（初期） |
| 5 | 約308 | 炭焼き窯 |
| 10 | 約1,000 | **釣り** |
| 15 | 約2,000 | 樹種：杉 |
| 30 | 約6,200 | **鉱石** |
| 35 | 約8,300 | 樹種：樫 |
| 50 | 約15,400 | **料理** |

- Lvアップごとに全生産 +1%（永続・軽ボーナス）。

## 2. 系統ごとの内容

### 2-1. 材木（Lv1〜）基礎供給源
| 項目 | 内容 |
|---|---|
| 採取 | 伐採：+1 原木/回（×斧Lv、×樹種倍率） |
| 加工 | 原木 5 → 板材 1 |
| 炭焼き（Lv5解放） | 板材 2 → 木炭 1 |
| 販売 | 板材 1 → 10G（×販路拡大） |
| 樹種 | 松 ×1（Lv1）／杉 ×2（Lv15）／樫 ×4（Lv35）※単価倍率 |
| 供給 | **木炭** → 鉱石の製錬・料理の火力へ |

アップグレード：鋭い斧 / 木こり(自動伐採) / 製材所(自動加工) / 炭焼き窯(自動炭化) / 行商人(自動販売) / 販路拡大(単価+5)

### 2-2. 釣り（Lv10）待ち＋ヒットの軽アクション
| 項目 | 内容 |
|---|---|
| 採取 | 竿を投げる→数秒後ヒット→タップで釣果。放置時は自動でそこそこ釣れる |
| 魚ランク（出現率） | 雑魚 5G/40% ・ 中型 20G/30% ・ 大物 80G/15% ・ レア 400G/10% ・ 超レア 5,000G/5% ※ルアーで上位率が上昇 |
| 加工 | 魚 1 → 干物 1（単価×2・保存性UP） |
| 販売 | 魚・干物 → G |
| 供給 | **魚** → 料理の主材料へ |
| 図鑑 | レア魚コレクション。集めると釣り効率+ボーナス |

アップグレード：釣り竿(釣果+) / ルアー(レア率+) / 釣り人(自動) / 生簀(在庫上限+) / 干物棚(自動加工)

### 2-3. 鉱石（Lv30）深度＋製錬
| 項目 | 内容 |
|---|---|
| 採取 | 採掘。**坑道の深度**で鉱石ランクUP、硬い（必要ヒット増） |
| 鉱石ランク | 鉄 → 銅 → 銀 → 金 → オリハルコン（生の鉱石は販売不可、要製錬） |
| インゴット売値 | 鉄100 / 銅200 / 銀300 / 金800 / オリハルコン2,000G |
| 宝石（レアドロップ） | ランク外。採掘中にまれに出土（売値1,500G前後、料理・店舗用素材） |
| 製錬 | 鉱石 3 ＋ **火力（ランク相応）** → インゴット 1 ← 材木の炭を消費 |
| 出口 | インゴット：販売 or 各系統の設備強化素材 |
| 供給 | **金属・道具** → 全系統の一部アップグレードに要求 |

アップグレード：ツルハシ(硬さ貫通) / 削岩機(自動採掘) / 溶鉱炉(自動製錬) / 坑道拡張(深度解放) / 選鉱(宝石率)

### 2-4. 料理（Lv50）集大成・最終出口
| 項目 | 内容 |
|---|---|
| 生産 | レシピ制。素材を組み合わせて完成品（最高単価） |
| レシピ例 | 焼き魚：魚1＋火力1 → 30G ／ 魚定食：干物2＋火力2 → 120G ／ 豪華コース：レア魚1＋宝石1＋火力5 → 3,000G |
| 消費 | 魚(釣り)＋火力(材木の炭)＋調理器具(鉱石のインゴット) |
| 販売 | 完成した料理を販売してG獲得（他系統と同じ販売方式。客数・席数の概念は無し） |

アップグレード：かまど(火力効率) / コック(自動調理) / レシピ本(解放) / 料理店(自動販売) / 看板(単価+)

## 3. 相互依存まとめ（後半が前半を消費し続ける）

```
材木 ──木炭──▶ 鉱石(製錬)・料理(火力)
釣り ──魚────▶ 料理(主材料)
鉱石 ──金属──▶ 全系統(設備強化・調理器具)
料理 ─────────▶ 最終収益（最大の売上）
```

- 後半系統ほど「複数素材を束ねて高単価」になり、前半系統の増産が常に必要。
- 一部の強力アップグレードは **お金＋素材（例：溶鉱炉Lv5に鉄インゴット×20）** を要求し、系統を跨いだ計画性を出す。

## 7. 現状バランスの所見（要調整）

- **評価軸は2つ**：G建ての利幅ではなく「時間あたりスループット」と「律速素材」で見る。→ 表に **時間/自動速度 列が必須**。
- **木炭が中〜終盤の共通ボトルネック**：鉱石製錬＋全料理が消費。木炭1=原木10と重いので、炭焼き窯の自動速度・樹種倍率を強めに。
- **インゴットの価値/用途が未定義**：売値と、設備強化での要求量（例：溶鉱炉Lv5＝鉄インゴット×20）を決める。
- **焼き魚が薄利（+5G）**：opportunity costでは薄いが、素材が自動なら律速のみ。入口レシピとして単価30→40Gに上げるか木炭消費を外す案も。
- **単価の桁飛び（10G→3,000G）は正常**：各系統内10倍刻みが効いている。

## 9. 燃料（炭）システム

炭は3グレード。下流は抽象単位「火力」で要求し、どの炭でも支払える。

### 炭グレードと生産
| 炭 | 火力 | 施設 | 入力→出力 | 解放 | 板材あたり火力 |
|---|---|---|---|---|---|
| オガ炭 | 1 | 炭焼き窯 | 板材2 → オガ炭1 | Lv5 | 0.5 |
| 白炭 | 3 | 白炭窯 | 杉の板材3 → 白炭1 | Lv20 | 1.0 |
| 備長炭 | 5 | 備長炭窯 | 樫の板材2 → 備長炭1 | Lv35 | 2.5 |

- 白炭窯：25,000 × 2.0（産出 白炭 0.8/秒 ×Lv）
- 備長炭窯：200,000 × 2.0（産出 備長炭 0.5/秒 ×Lv）
- 高グレードほど板材効率は良いが、杉・樫（高級木＝売っても高い）を食う → 「炭にするか売るか」の判断。

### 下流の火力要求
| 用途 | 火力 |
|---|---|
| 鉄・銅 製錬 | 1 |
| 銀 製錬 | 2 |
| 金 製錬 | 3 |
| オリハルコン 製錬 | 5 |
| 焼き魚 | 1 |
| 魚定食 | 2 |
| 豪華コース | 5 |

- 高火力の用途（オリハルコン製錬・豪華コース＝火力5）は備長炭1個で足りる＝処理回数を食わずスループット向上。低グレードだと5個必要でボトルネック化。
- **支払いルール**：余り火力が最小になる組み合わせを自動選択、端数はオガ炭で調整（例：火力5→備長炭1を優先）。
